Request Header Parser Microservice

- FreeCodeCamp Backend Challenge

A microservice which reads your request headers and returns JSON containing your client IP address, language and software.

User Story : I can get the IP address, language and operating system for my browser.

Example Usage :

https://fcc-hdr-parser.glitch.me/whoami

Example Output :

{ "ip":"157.50.13.255", "software":"Mozilla/5.0 (Windows NT 6.3; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/63.0.3239.84 Safari/537.36", "language":"en-US,en" }
